Projet SAÉ 2.03

Introduction à GIT et Déploiement avec Docker

Module 1 : Maîtriser GIT

TP 1 : Répertoire Local

  • Configuration de l'environnement GIT (identifiant, éditeur).
  • Création et initialisation d'un dépôt local.
  • Gestion des états : Modified, Staged, Committed.
  • Création du fichier .gitignore pour un projet Java.

TP 2 : Dépôt Distant (GitHub)

  • Création et configuration d'un compte GitHub.
  • Sécurisation des échanges via clé SSH.
  • Synchronisation entre dépôt local et distant (push/pull).
  • Clonage d'un projet distant.

TP 3 : Travail en Équipe & Branches

  • Ajout de collaborateurs sur un dépôt.
  • Développement collaboratif (Projet CryptoMonnaie).
  • Création et gestion des branches pour les nouvelles fonctionnalités.
  • Fusion (merge) des branches dans la branche principale.

TP 4 : Conflits & Pull Requests

  • Compréhension et génération de conflits de fusion.
  • Résolution manuelle des conflits dans les fichiers.
  • Initiation au concept de "Pull Request" pour l'open-source.

Module 2 : Conteneurisation avec Docker

Bases de Docker

  • Différence entre une Image et un Conteneur.
  • Commandes essentielles (run, ps, stop, rm, exec).
  • Exposition des ports pour les services web (ex: Apache httpd).
  • Gestion des volumes et copie de fichiers vers un conteneur.

Création d'Images (Dockerfile)

  • Écriture d'un Dockerfile personnalisé.
  • Construction d'images avec docker build.
  • Installation de services depuis une image debian vierge.
  • Déploiement rapide via GitHub et Docker.
